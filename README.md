# Dotfiles

My personal dotfiles for arch.

---

## Dependencies

- `hyprland`
- `waybar`
- `nvim`
- `kitty`
- `fastfetch`
- `btop` 
- `rofi`


---

## Installation

### First clone the repo
```bash
git clone https://github.com/Jhonnikek/dotfiles.git
cd dotfiles
```

### Run the installation file
```bash
./install.sh
```
