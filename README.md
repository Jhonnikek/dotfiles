# Dotfiles

My personal dotfiles for arch.

---

## Dependencies

- `alacritty`
- `fastfetch`
- `btop` 
- `rofi`
- `ttf-fira-sans`
- `ttf-fira-code`
- `ttf-firacode-nerd`

---

## Installation

### First clone the repo
```bash
git clone https://github.com/yourusername/dotfiles.git
cd dotfiles
```

### Run the installation file
```bash
./install.sh
```
